# Sales Data Analysis Dashboard

### Author: Dhruvil Shah

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover meaningful business insights and create a sales dashboard using Python.

The objective was to analyze sales performance, identify trends, understand customer behavior, and generate actionable insights through visual analytics.

---

## Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

---

## Project Workflow

1. Data Loading and Initial Exploration
2. Missing Value Investigation
3. Data Cleaning
4. Feature Engineering
5. Sales Trend Analysis
6. Regional and Category Analysis
7. Customer & Product Insights
8. Profitability Analysis
9. Dashboard Visualization

---

## Feature Engineering

Additional features were created from date columns:

- Month extraction
- Year extraction

These features enabled time-based business analysis and seasonal trend investigation.

---

## Dashboard Visualizations

### Monthly Sales Trend
Analyzed monthly sales patterns to identify seasonality and fluctuations.

### Sales by Region
Compared geographical performance across business regions.

### Sales by Category
Evaluated product category contributions to overall revenue.

### Top Selling Products
Identified products generating the highest sales.

### Profit by Category
Compared sales performance against actual profitability.

---

## Custom Exploratory Analysis

Additional business-focused analysis was performed beyond standard dashboard requirements:

### Discount vs Profit Analysis
Investigated whether aggressive discounting negatively impacts profitability.

### Monthly Category Sales Heatmap
Analyzed category performance across different months to identify seasonal demand patterns.

These analyses were added to derive deeper business insights and move beyond standard reporting.

---

## Key Insights

- Sales performance varied significantly across regions.
- Certain product categories generated disproportionately high revenue.
- Higher sales did not always translate into higher profits.
- Discounting showed potential impact on profitability.
- Seasonal patterns emerged across categories and months.
- A small number of products contributed heavily to total sales.
